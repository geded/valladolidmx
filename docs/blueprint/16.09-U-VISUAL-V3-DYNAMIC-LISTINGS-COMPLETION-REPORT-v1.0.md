## §1 · Objetivo

Unificar los 8 listados de descubrimiento bajo un único sistema visual y de UX, alineado al **Founder Discovery Principle**: cada listado responde en segundos qué se descubre, cuál es la mejor opción, qué hay cerca, por qué es diferente y qué abrir ahora. Cero cambios de rutas, contratos, tipos internos, backend ni SEO.
