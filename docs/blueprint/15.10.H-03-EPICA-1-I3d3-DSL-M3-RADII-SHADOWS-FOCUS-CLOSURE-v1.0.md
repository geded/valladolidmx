## Cambios
### `src/styles.css`
- **Token nuevo (aditivo):** `--radius-pill: 9999px` dentro de `@theme inline`.
- **6 `@utility` DSL nuevas** (todas reutilizan tokens existentes):
  - `rounded-pill` → `border-radius: var(--radius-pill)`
  - `shadow-soft` → `var(--shadow-card)`
  - `shadow-elevated` → `var(--shadow-raised)`
  - `shadow-floating` → `var(--shadow-overlay)`
  - `ring-focus` → `var(--shadow-focus)` + `outline: none` (accesible, respeta `--ring`)
  - `border-hairline` → `1px` sobre `color-mix(--border 60%)`
  - `border-subtle` → `1px` sobre `--border`

## Founder Decisions
- **Safe Visual Migration Rule:** additive-first, sub-ola independiente, migración componente-a-componente en olas posteriores (M4..).
- **Freeze de infraestructura:** no se crean engines, providers, ni se redefinen tokens ya existentes.
- **Single Source of Truth visual:** DSL v1.0 sigue siendo la única fuente de verdad; M3 lo *expone* como utilidades.
- **Visual Identity Rule:** foco accesible + sombras cálidas refuerzan la esencia colonial.
