# CV5 · Intelligent Travel Workspace — Visión Oficial v1.0

**Estado:** Aprobado como visión. **No implementar** hasta autorización explícita del Founder.
**Depende de:** CV0 Travel Plan Contract, CV2 Bridge, CV3 Drawer, CV1 Panel Admin, CV4 Sales Journey.

## Principio Fundador

El viajero nunca deberá sentir que cambia de herramienta. Siempre estará dentro de **Mi Viaje**. El Itinerario es únicamente una vista del Travel Plan. **Prohibido crear un segundo modelo de viaje.** Fuente única: `travel_plans` + `travel_plan_items`.

## Evolución

- **Fase 1 (construida):** Inspiración → Agregar a Mi Viaje → Travel Plan → Alux → Concierge → Checkout.
- **Fase 2 (CV5):** Travel Plan → Mi Viaje → Resumen → Itinerario Inteligente → Reservas → Concierge → Alux → Documentos → Recuerdos. Todo pertenece al mismo objeto.

## Mi Viaje — vistas del mismo Travel Plan

Resumen · Itinerario · Reservas · Concierge · Alux · Documentos · Recuerdos. No son módulos separados.

## Itinerario Inteligente

Dinámico: mover actividades, cambiar horarios, reorganizar días, eliminar y agregar. Cada cambio sincroniza Travel Plan, Concierge, Alux, Timeline, Email, PDF y link compartido. **Una sola versión del viaje.**

## Optimización

Alux propone reorganizaciones considerando distancias, tráfico, horarios, clima, duración, presupuesto, fatiga, gastronomía, amanecer/atardecer, días cerrados, eventos. El viajero siempre decide.

## Roles

Viajero **decide** → Alux **propone** → Concierge **perfecciona**.

## Mapas

Cada actividad conoce ubicación, tiempo estimado, ruta, distancia. Vistas Lista · Mapa · Mixto. Reutiliza Google Maps.

## Notificaciones

"Es momento de salir", "Tu restaurante abre en 30 minutos", "El Concierge actualizó tu itinerario", "El tráfico aumentó".

## Founder Memory Principle

El usuario nunca reinicia. Travel Plan, Alux, Concierge, Checkout, Timeline y Mi Viaje comparten el mismo contexto.

## Acceso Rápido

Acceso permanente **❤️ Mi Viaje** desde cualquier pantalla. Si hay viaje activo: CTA destacado "Tienes un viaje en curso · Abrir Mi Viaje".

## Product Vision

La experiencia principal del viajero en ValladolidMX es **Mi Viaje** — espacio personal del viajero. Ciclo: Inspiración → Travel Plan → Alux → Concierge → Checkout → Itinerario Inteligente → Viaje en curso → Recuerdos → Próximo viaje. Toda nueva capacidad se integra dentro de Mi Viaje.

## Roadmap

**No implementar ahora.** Cuando el Founder autorice: épicas pequeñas reutilizando Travel Plan, Context Engine, Alux, Concierge, Navigation Contract, Google Maps y Travel Workspace. Prohibidos nuevos motores o modelos paralelos.
