## 1. Alcance entregado

Consumo del bloque oficial `vmx.experience.related-collection` en la
superficie territorial **Categoría en Destino**
(`/oriente-maya/:destino/:categoria`). Toda página categoría publicada
cierra ahora con una sección **"Sigue descubriendo"** que ofrece 2
caminos de descubrimiento explicables (Explainable by Default).

### Grupos ofrecidos por el bloque
| id | Fuente | Rationale |
|---|---|---|
| `otras-categorias-destino` | server fn `getCategoryRelated` — empresas de OTRAS categorías del MISMO destino | "Otras categorías en {destino}" |
| `misma-categoria-otros-destinos` | server fn `getCategoryRelated` — empresas de la MISMA categoría en OTROS destinos del Oriente Maya | "{Categoría} en otros destinos del Oriente Maya" |

Guard silencioso: si ambos buckets vienen vacíos la sección no se
renderiza (nunca se muestran encabezados huérfanos).
