## 1. Alcance entregado
- Server fn pública `aluxContextualSuggest` en
  `src/lib/alux/contextual-suggest.functions.ts`:
  · Input Zod estricto sobre el snapshot del Context Engine
    (region/destination/category/business/product/limit).
  · Guard `region.slug === "oriente-maya"` + `destination.slug` obligatorio.
  · Recuperación determinista sobre `businesses` publicados (misma policy
    pública que Discovery); excluye la empresa actual.
  · Ranking según nivel de contexto:
    · Product/Business → hermanos de misma categoría primero, luego otras.
    · Category → publicados en la categoría.
    · Destination → rotación restaurantes → experiencias → hoteles → otras.
  · Rationale humano y explicable por item (Política 06).
  · DTO estable `{ suggestions[], contextSnapshot, generatedAt, reason }`.
- Consumo en `AluxFloatingTrigger`: `useServerFn` + `useQuery`
  (key = tupla del snapshot, `staleTime 60s`, `enabled` sólo con Sheet abierto
  y destino resuelto). Loading state coherente; fallback al `ctx.related` del
  Context Engine cuando la fn devuelve `[]`.
