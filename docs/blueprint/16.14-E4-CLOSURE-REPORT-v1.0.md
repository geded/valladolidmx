## Impacto de producto

`/arma-tu-viaje` es hoy el **Planificador Oficial del Oriente Maya**:
hero productivo con 4 estados, preview del guest-queue, expediente
activo real, compartir vía link público con Imprimir/PDF, y copiloto
Alux Traveler explicable integrado en la misma superficie. El bucle
**Discovery → Expediente → Copiloto → Concierge** queda operativo sin
abrir infraestructura paralela.

- Persistencia: `travel_plans` + `travel_plan_items` (sin cambios de
  esquema salvo columnas `share_token` / `shared_at` en E4.3).
- Contexto: Context Engine + `useAluxContext`.
- Navegación: Navigation Contract v1.0 (`/oriente-maya/...`).
- IA: `alux_traveler_*` + `AluxTravelerPanel` (Sub-ola H).
- Guest queue: `src/lib/traveler/guest-queue.ts`.
- Cero engines nuevos, cero rutas paralelas, cero motores AI adicionales.
