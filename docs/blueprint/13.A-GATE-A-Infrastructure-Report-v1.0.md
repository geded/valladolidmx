## 1. Resumen ejecutivo

El Bloque A deja operativa la **plataforma sobre la que se construye toda la
Fase 1**: base de datos PostgreSQL, Storage con 8 buckets gobernados,
autenticación (email + Google) y los cimientos técnicos (extensiones,
helpers, convención de migraciones) que usarán los Bloques B–D.

No se ha creado todavía ninguna tabla de negocio — eso corresponde al
Bloque B (Domain Model).

Override admin se añadirá en Bloque C cuando exista `has_role()`.

### 2.4 Base de datos – Fundación

**Extensiones** (schema dedicado `extensions`):

- `unaccent` – normalización de slugs y búsqueda.
- `pg_trgm`  – índices trigram / fuzzy search.
- `citext`   – texto case-insensitive.
- ❌ `pgcrypto` descartado: `gen_random_uuid()` es nativo en PG13+.

**Helper compartido**:

- `public.set_updated_at()` – trigger genérico para `updated_at`.

**`search_path`** ajustado a `"$user", public, extensions` para que las
funciones de extensiones sean accesibles sin schema-qualifying.

### 2.5 Autenticación (13.3 §3, §7)

| Configuración                 | Valor          | Justificación                                   |
| ----------------------------- | -------------- | ----------------------------------------------- |
| Email/password                | ✅ habilitado  | Método base 13.3 §3.                            |
| Confirmación por email        | ✅ requerida   | 13.3 §7 "Verificación por correo".              |
| Sign-up abierto               | ✅ habilitado  | Visitante puede registrarse (13.3 §2).          |
| Anonymous users               | ❌ deshabilitado | 13.3 §9 "no privilegios por defecto".         |
| Password HIBP check           | ✅ activo      | Mejor práctica de seguridad.                    |
| Google OAuth (managed)        | ✅ habilitado  | 13.3 §3 "Google".                               |
| Apple OAuth                   | ⏳ diferido    | Se habilita junto a UI de auth (Bloque C/UI).   |

Sign in con Google se realiza vía `lovable.auth.signInWithOAuth("google", …)`
(módulo `@lovable.dev/cloud-auth-js` instalado).
