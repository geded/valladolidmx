## 2. Contratos invariantes
1. Toda fuente de contexto del destino se integra como Contributor. Prohibidas rutas alternativas.
2. Los Contributors publican al Context Engine; superficies leen únicamente del Context Engine.
3. Cada señal declara `source`, `at`, `ttlMs`, `scope`, `payload`, `explain` (Explainable by Default).
4. Registrar un nuevo Contributor NO requiere refactor de consumidores.

## 3. Founder Experience First — declaración obligatoria
- **Cuándo aporta valor:** cuando una superficie (Now·Next·Later, Alux, Mi Viaje, Concierge) requiere consciencia operativa del destino para decidir.
- **Auto-ocultamiento:** una señal con TTL vencido se descarta; una superficie sin señales activas de su interés no renderiza contexto.
- **Acción concreta:** cada señal expone `explain.rationale` que las superficies convierten en decisión (cambiar plan por lluvia, salir antes por tráfico, escoger negocio abierto ahora).
