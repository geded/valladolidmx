# Activos Oficiales de Marca — Valladolid.mx

Esta carpeta es la **única fuente oficial de activos de marca** del proyecto Valladolid.mx.

## Propósito

Centralizar y proteger todos los recursos gráficos oficiales que constituyen la identidad visual de la marca:

- **logos/** — Logotipo, isotipo, imagotipo y variantes oficiales (color, blanco, monocromo).
- **photography/** — Banco fotográfico oficial organizado por uso:
  - `hero/` — Fotografías para el Hero principal y secciones de alto impacto.
  - `destinations/` — Fotografías de destinos turísticos.
  - `categories/` — Fotografías para categorías de experiencias.
  - `businesses/` — Fotografías de portada, galería y producto de empresas afiliadas.
- **icons/** — Iconografía oficial, sellos institucionales y recursos vectoriales.
- **colors/** — Definiciones oficiales de la paleta cromática institucional.
- **fonts/** — Archivos de tipografía oficial autorizada para uso en la plataforma.
- **social/** — Imágenes y recursos para redes sociales, Open Graph y materiales promocionales.

## Regla obligatoria

> **Ningún desarrollador, diseñador o herramienta de IA debe crear activos alternativos cuando los oficiales no existan.**

Si un activo oficial aún no está disponible:

1. Dejar el componente preparado para recibir el activo oficial.
2. Utilizar un placeholder temporal claramente identificado como provisorio.
3. No reinterpretar, rediseñar ni generar por IA ningún elemento de identidad oficial.

Esta regla aplica a: logotipos, isotipos, favicons, símbolos, colores institucionales, tipografías oficiales, fotografías, videos, iconografía y sellos institucionales.
