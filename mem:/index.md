# Project Memory

## Core
Founder Experience First Rule (CV6, vinculante): toda capacidad Live Destination Companion debe responder "¿qué ayuda concreta recibe el viajero ahora?". Prioriza recomendaciones accionables sobre información pasiva; se oculta automáticamente cuando no aporta valor. Ver `mem://policies/founder-experience-first.md`.
Founder Decision Center Principle (CV6.2, vinculante): Now·Next·Later es Centro de Decisiones, no timeline. Jerarquía por tarjeta: acción → motivo → contexto → secundarias. Priorización automática por impacto experiencial. Auto-hide obligatorio. Ver `mem://policies/founder-decision-center.md`.

## Memories
- [Founder Experience First](mem://policies/founder-experience-first.md) — CV6: capacidades sólo si aportan ayuda concreta y accionable; auto-ocultamiento cuando no aportan valor.
- [Founder Decision Center](mem://policies/founder-decision-center.md) — CV6.2: Now·Next·Later como centro de decisiones; jerarquía acción-primero; priorización por impacto; auto-hide.
