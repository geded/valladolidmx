# 09-UX-UI-DESIGN-SYSTEM.md

# UX / UI Design System
Versión 1.0

## Objetivo

Definir la experiencia visual y los componentes reutilizables de toda la plataforma.

---

# Filosofía

No copiar el diseño antiguo de Valladolid.mx.

Conservar su identidad y evolucionarla con una experiencia moderna.

---

# Principios UX

- Mobile First.
- PWA.
- Navegación intuitiva.
- Máximo tres clics para llegar a cualquier contenido.
- Fotografías protagonistas.
- Acciones claras.
- Alux integrado de forma natural.

---

# Componentes globales

- Header
- Footer
- Hero
- Breadcrumb
- Buscador
- Cards
- Carruseles
- Filtros
- Galerías
- CTA
- Formularios
- Modales
- Reseñas
- Alux Tips
- Arma tu Viaje

Todos reutilizables.

---

# Pantalla Home

Debe incluir:

- Hero con imagen impactante del Oriente Maya.
- Eslogan:
  "Despierta en Valladolid y descubre el Oriente Maya de Yucatán."
- Buscador principal.
- Destinos destacados.
- Categorías.
- Experiencias.
- Recomendaciones de Alux.
- CTA de Arma tu Viaje.
- Empresas destacadas.
- Reseñas.

---

# Micrositios

Cada destino debe tener:

- Hero propio.
- Historia.
- Mapa.
- Qué hacer.
- Hoteles.
- Restaurantes.
- Experiencias.
- Eventos.
- Empresas.
- Consejos de Alux.
- Breadcrumb permanente.

---

# Fichas de empresa

Mostrar:

- Portada.
- Galería.
- Video.
- Descripción.
- Servicios.
- Horarios.
- Ubicación.
- Contacto.
- Reseñas.
- Recomendaciones de Alux.
- Botón Agregar a Arma tu Viaje.

---

# Arma tu Viaje

Debe ser accesible desde cualquier pantalla.

No como carrito.

Como panel de planificación.

Debe permitir:

- agregar
- eliminar
- ordenar
- comentar
- enviar al concierge

---

# Alux

No debe mostrarse únicamente como chat.

También mediante:

- Consejos.
- Recomendaciones.
- Tarjetas.
- Resúmenes.
- Sugerencias contextuales.

---

# Responsive

Optimizar para:

- teléfono
- tablet
- escritorio

---

# Accesibilidad

- Alto contraste.
- Navegación por teclado.
- Tamaños táctiles adecuados.
- Textos alternativos.

---

# Animaciones

Suaves.

Nunca distraer.

Priorizar rendimiento.

---

# Identidad visual

Conservar:

- Imagen hero.
- Sensación de portal regional.
- Fotografía protagonista.

Modernizar:

- Tipografía.
- Grid.
- Espaciados.
- Componentes.
- Interacciones.

---

Variables globales:

- Colores.

Todos reutilizables.

# Oriente Maya EN VIVO

Esta sección sustituye "Empresas Destacadas".

Objetivo:

Mostrar que el destino está vivo utilizando contenido oficial.

## Fuente

- Instagram oficial de Valladolid.mx
- Facebook oficial de Valladolid.mx

La información debe obtenerse mediante APIs oficiales, nunca mediante capturas o simulaciones.

## Contenido

- Fotografías
- Reels
- Videos
- Publicaciones destacadas

Desde el CMS se podrá definir:

- Qué cuentas conectar
- Cuántas publicaciones mostrar
- Frecuencia de actualización
- Publicaciones fijadas
- Relación con destinos

Al seleccionar una publicación podrá abrirse:
- La publicación original.
- El destino o micrositio relacionado.

---

# Reglas para Lovable

- Construir biblioteca de componentes.
- No duplicar pantallas.
- Reutilizar layouts.
- Todo conectado al CMS.
- Preparar modo oscuro en el futuro.

---

# Criterios de aceptación

El usuario reconoce la esencia de Valladolid.mx pero percibe una plataforma completamente moderna, rápida, elegante y preparada para crecer.
